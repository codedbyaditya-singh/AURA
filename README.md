# 🎙️ AURA – AI Powered Voice Assistant

An intelligent desktop voice assistant built with Python that listens, understands, and performs real-world tasks using speech recognition, wake-word detection, and natural language processing.

⸻

# 🚀 Overview

AURA is a modular AI-powered desktop voice assistant designed to interact with users through natural voice commands. It combines wake-word detection, speech recognition, intent understanding, and text-to-speech to provide a seamless hands-free experience.

The assistant can automate everyday tasks such as retrieving weather updates, playing YouTube content, reading news headlines, sending WhatsApp messages, opening websites, managing reminders, and responding conversationally—all through voice commands.

Built with scalability in mind, AURA follows a modular architecture, making it easy to integrate additional commands and AI capabilities.

⸻

# ⚙️ How It Works

1. The assistant continuously listens for the wake word " HEY AURA ".
2. Once activated, it records the user’s voice command.
3. Speech is converted into text.
4. The intent classification module identifies the user’s request.
5. The corresponding action module executes the task.
6. AURA responds using text-to-speech.
